Why YOU should use WSL for your programming




WSL stands for Windows Subsystem for Linux, and it allows you to boot Linux terminals onto your Windows machine. WSL supports a variety of Linux distributions, such as:

  • Ubuntu (Most Common)
  • Debian (Root of Ubuntu, less support)
  • Kali Linux (Different commands, less support)
  • And some more. Best for beginners is Ubuntu, with the most support and easiest interface.

To install WSL, there are two methods, both of which are incredibly easy.

Method 1

One way to install WSL is through Microsoft Store, the builtin app installer.

First go to Microsoft store and type in Ubuntu (preferably, although you could use other distros).
